Technical info
There are two Vue components implemented, one for single item selection and 
other for multiple items.

Some configuration can be passed to the components as props, the options 
displayed, the empty selection text, the done button text and the initial 
selected values.

The options and selected objects need to have at least:

    - id: for identification of the object
    - title: to be displayed in the options list

There is some room for improvements. For example, in the image in Google Drive
document all the options seems selected but the "All Arrangement Types" is not,
my interpretation, if this item is used, all the options are selected, but if
all the options are selected, this option should be also selected. Maybe there 
is some specification not in that document.

There is also pending to implement some mechanism to get the values selected from 
the components, there are options like emit an event from the component or use a 
more complex state management library like Vuex.
